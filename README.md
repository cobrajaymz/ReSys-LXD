# ReSys LXD Project

This repository contains scripts and configuration files to set up the ReSys multi-tier container infrastructure using LXD on Ubuntu.

## Scripts

- **install_lxd.sh** – Install and initialize LXD.
- **setup_infra.sh** – Create networks, profiles, and launch containers.
- **configure_containers.sh** – Configure static IPs, SSH keys, and /etc/hosts for all containers.
